# Local J_OS Project Link (Example)

Copy this file to `.j_os/project-link.local.md` in your local project workspace, customize the values, and ensure it remains untracked by adding it to `.gitignore`.

## Configuration Fields:
*   `project_id`: The lowercase ID of the project matching the registry (e.g. `inkspire`).
*   `j_os_root`: The absolute path to the local copy of the J_OS vault (e.g. `J:/My Drive/Master_Vault/J_OS`).
*   `bridge_mode`: Set to `standard` for normal operation.
*   `status`: Marks this file as `example`.
