# Daily Logs

Use this folder for dated notes about meaningful work.

Recommended filename:

```text
YYYY-MM-DD.md
```

Useful headings:

- Session Purpose
- Decisions
- Files Changed
- Validation
- Artifacts
- Follow-Ups

Keep daily logs short. Their job is to help the next session understand what changed and why.
