# Subagents MOC (The Registry)

This document catalogs the native subagents that the primary AI can dynamically define and invoke. 

- [[Code_Reviewer]]: Analyzes code diffs across 5 axes (Correctness, Readability, Architecture, Security, Performance). Participates in the Safe Refactor Protocol.
- [[Architect]]: Analyzes the broader system architecture and drafts Implementation Plans using Spec-Driven Development (SDD).
- [[UI_Designer]]: Focuses entirely on CSS, aesthetics, and creating UI mockups using the project's Design System.
- [[Vault_Keeper]]: Administrative file-syncing helper that archives transient plans into the Vault's chronological folder structure.
